# alpha.10.7.7 自适应事实抓取补充

事实链保持不变，但调用粒度调整为：正文准入 → 小批事实提取 → 失败退避/拆批 → 表格提交 → 事件与世界书。504 不再对同一大批次立即重放；多轮失败优先拆小，单轮失败才有限重试。历史重建按 4–10 条规划，并继续由字符预算二次切分。

提示词总量不再随存档无限增长：事实提取优先发送焦点、基础设定、手工锁定、当前活跃和本轮提及行；阶段总结与大总结只发送高优先历史片段。被省略内容仍由代码完整保存，不会因未发送给模型而删除。

# alpha.10.7.6 模型路由补充

```text
当前正文连接 ── generateRaw ── 当前活动模型

指定 Connection Profile ── sendRequest(profileId) ── Profile 保存的 API / 模型 / 密钥引用 / 代理
```

- 两条通道互不隐式回退。
- Profile 任务创建时冻结 ID 与路由指纹，执行前再次校验。
- 不同 Profile 使用独立异步 lane；同一 Profile 内保持顺序。
- 对照诊断用于区分当前连接故障、Profile 独有故障和共享上游故障。
- 不执行 `/profile`，不写入 `selectedProfile`，不改变正文当前模型。

# Alpha 10.7 架构契约

## 唯一事实源与准入

聊天中实际保存的 AI 最终正文仍是唯一事实源。审核可关闭；无论采用 `audit`、`revision`、`bypass`、`history-rebuild` 或 `history-cache` 准入模式，只有与当前正文指纹一致的最终正文才能进入记忆链。

## 严格派生方向

```text
final body
  → fact package
  → active tables
  → event operations + stage overview
  → derived event registry
  → long-term records + event sedimentation
  → lorebook documents
```

上层不得反向改写下层：

- 事件总结不能改写正文；
- 大总结不能覆盖当前表格；
- 世界书不能成为事实来源；
- 关系图谱不能写回任何业务数据。

## 活跃表格与事件记忆的职责

活跃表格保存当前高频、明确、不能轻易忘记的状态。事件记忆保存已经发生的因果对象、传播、痕迹、持续影响和逐步降低的历史精度。

同一事实不应在两个层级争夺当前状态：表格负责“现在是什么”，事件条目负责“发生过什么以及留下了什么”。

## 事件注册表是派生缓存

`chatState.eventEntries` 不作为独立事实源。它由以下内容按时间顺序重建：

1. 每份小总结的 `eventOperations`；
2. 每份大总结的 `eventUpdates`。

正文编辑、删除或 Swipe 使总结依赖失效时，受影响的小总结和后续大总结先被移除，然后重新计算事件注册表。

## 事件精度

精度顺序固定为：

```text
exact → compressed → fuzzy → trace
```

没有新的来源事实 ID 时，更新只能保持或降低精度。新的最终正文出现明确确认时，可以重新提高精度。

大总结的 `event_update` 只能保持或降低精度，并可提供与新精度一致的替代事实文本。它不能读取原始正文或表格，因此不能从上游之外恢复旧细节。

## 传播和痕迹

传播字段只记录已经在正文显影的内容：

- 传播范围；
- 已知者；
- 渠道；
- 误传或版本偏差；
- 记录、物证、传言或流程痕迹；
- 持续影响。

插件不按时间自动生成传播变化，也不保存下一次复核时间。

## 条目级发布策略

每个事件条目具有独立 `activation`：

- `constant`：只允许重要、仍活跃或未决、且精度不低于 compressed 的少量事件；
- `keyword`：默认触发方式；
- `vector`：fuzzy、trace 和语义相关历史。

世界书发布时，阶段概览不再作为整块提示词条目发布。事件注册表中的每个事件单独生成世界书文档。

## 删除保护

事件只有同时满足以下条件才可删除：

- 重要度为 `low`；
- 状态为 `trace`；
- 精度为 `trace`；
- 没有持续影响。

不满足条件的 remove 请求被忽略，避免模型一次错误输出造成长期记忆丢失。

## 异步与 UI

审核仍是正文发布前的阻断任务。表格、事件总结、大总结和世界书均为非阻塞后台任务。

前端显示事件注册表数量、状态、精度、传播范围和常驻/关键词/向量分布。关系图谱继续只读当前表格。
