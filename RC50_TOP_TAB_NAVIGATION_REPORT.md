# Mirror Abyss 1.2.0-rc.50 顶部滑动导航报告

## 修改目标

rc.49 工作区在桌面使用左侧栏，在手机使用侧滑抽屉。功能数量增加后，侧栏占用正文宽度，手机还需要先打开菜单再切换页面。本版恢复更直接的顶部标签方式，并沿用现有工作区内容、状态卡与移动端对象卡片。

## 实际修改

- 删除工作区左侧栏、侧栏状态、侧栏页脚、手机菜单按钮、抽屉关闭按钮和遮罩。
- 工作区改为单列全宽结构：页头、横向标签栏、内容区。
- 九个页面入口全部保留：总览、对象视图、对象图谱、分层总结、表格管理、规则审核、世界书、模型与设置、诊断。
- 标签栏使用横向自然宽度，不压缩、不换行，支持鼠标滚动条和手机触控左右滑动。
- 活动标签切换后自动滚入可视区域。
- 桌面、平板和手机统一一套导航逻辑，不再维护抽屉打开状态。
- 页头保留镜渊标识、当前页面、当前聊天状态、版本号和关闭按钮；手机宽度下自动压缩非必要文字。

## 保留边界

- 不改审核、状态提取、总结、世界书、任务队列和历史恢复逻辑。
- 不改 rc.49 兼容模式。
- 不删除任何页面或操作入口。
- 保留手机动态可视高度、安全区、对象表卡片化和正文极简状态条。

## 验证

- 更新 rc.45、rc.47、rc.48 UI 合约测试以匹配新导航。
- 新增 `rc50-top-tabs-ui` 专项测试。
- 完整 `npm run verify` 通过。
- TypeScript 类型检查、浏览器构建和最终 JavaScript 语法检查通过。
