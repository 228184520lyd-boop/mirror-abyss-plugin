# Mirror Abyss 1.3.3

发布日期：2026-07-21  
流水线：`ma-pipeline-67`

本版本集中修复“物品漏记、角色/全局误归档、没有场景条目”三类对象分流问题。

## 物品

物品不再要求多次出现或发生不可逆变化。只要物品可区分，且其所有权、位置、数量、完整性、可用性、隐藏状态、证据/任务用途、独特功能或后续能否取得会影响连续性，就应独立建档。多个命名物品不得合并成一条。

## 场景与地点

- 场景：本次真实发生的局面切片，记录参与者、冲突、限制、环境状态与承接点。
- 地点：离开当前场景后仍成立的空间本体、功能、布局、归属和访问条件。
- 时空：当前时间与所在位置。

旧设置会自动增加“场景”表。模型漏写当前场景时，插件从当前时空生成最小场景条目，避免场景表为空。

## 语义归档

每个状态条目都带有语义 `kind`。插件先按对象本质校验归属，再保存：具体个体进入角色；组织、阵营、政权、机构、群体格局、制度执行与世界态势进入全局。自动条目被确认放错表时会保留稳定 ID 后迁移；人工或锁定条目不自动搬动。

## 边界

本版本不改变审核、定向修正、两级总结和历史恢复流程。
