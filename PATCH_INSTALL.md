# 镜渊 1.1.0-rc.4.2 覆盖安装

本包覆盖现有 `rc.3`、`rc.4` 或 `rc.4.1` 安装。它依赖原插件目录中的核心文件：

- `index.js`
- `style.css`

压缩包故意不包含这两个文件，避免覆盖完整核心。

## 安装步骤

1. 打开现有 `mirror-abyss-plugin` 目录。
2. 将本压缩包内全部文件复制进去并允许覆盖。
3. 确认原目录中的 `index.js` 和 `style.css` 仍然存在。
4. 完整关闭并重新打开 SillyTavern 页面，或进行一次完整刷新。
5. 不要清缓存，不要删除镜渊数据。

## 预期设置

旧默认值会迁移为：

- 小总结：7 回合。
- 大总结：4 个小总结。
- 单批预算：5,000 字符，此项由代码固定，不需要界面手工输入。

用户自己设置的其他回合数不会被覆盖。

## 测试顺序

1. 先执行一次小总结。
2. 确认不再出现 `undefined (reading 'get')`。
3. 确认不再频繁出现“引用了非本批来源ID”。
4. 连续形成 4 个小总结后观察大总结。
5. 最后确认世界书同步没有重复条目。

失败时保留任务日志，不要连续重复点击，也不要清理现有记忆。

## 回滚

使用原 `rc.4.1` 覆盖包重新覆盖即可。回滚不要求清除任何数据。
