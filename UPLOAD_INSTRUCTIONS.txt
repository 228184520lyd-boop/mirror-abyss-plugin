Mirror Abyss 1.3.15 GitHub 网页上传版

1. 解压本文件。
2. 打开 GitHub 仓库根目录。
3. 选择 Add file → Upload files。
4. 一次上传解压后的全部文件并覆盖同名文件。
5. 该包已经把运行文件平铺并重写导入路径；不要再建立 bootstrap、core、domain 等子目录。
6. 提交后确认仓库根目录存在 manifest.json、index.js、bootstrap-app.js、domain-state-text.js。

此包用于 GitHub 网页上传，不用于在本地还原源代码目录。SillyTavern 可从该仓库安装。
