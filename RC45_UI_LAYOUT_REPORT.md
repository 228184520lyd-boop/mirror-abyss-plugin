# Mirror Abyss rc.45 UI 布局与样式重构报告

版本：`1.2.0-rc.45`  
流水线：`ma-pipeline-47`

## 重构范围

本轮保留所有业务入口、data 属性、任务动作和存储结构，只重构工作区的信息架构、DOM 布局和 CSS 样式系统。

## 桌面布局

- 工作区由顶部标签栏改为“左侧导航 + 右侧主内容”。
- 导航按“运行 / 管理 / 系统”分组。
- 主内容拥有独立页头，显示当前页面标题与职责说明。
- 内容区独立滚动，导航与页头不会随长表格、诊断或设置页面消失。
- 工作区最大宽度提升到 1440px，表格与图谱获得更完整的横向空间。

## 移动布局

- 900px 以下切换为单栏结构。
- 左侧导航变为顶部横向滚动导航，不使用抽屉脚本，避免额外开关状态。
- 页面标题保持固定，内容独立滚动。
- 工具栏、主要操作、连接配置、表格管理和编辑器自动降为单列。
- 620px 以下对象编辑器使用全屏模式。

## 视觉层级

- 建立统一 surface、border、accent、muted、radius 与 shadow 变量。
- 降低重复边框密度，区分背景、面板、卡片和输入层级。
- 主要操作使用主题强调色；危险操作保持独立警示边界。
- 阶段卡片、任务状态、历史恢复、审核错误分别使用明确状态色。
- 表格表头保持吸顶，行悬停与单元格层级统一。
- 支持 `prefers-reduced-motion`。

## 设置入口

- SillyTavern 扩展设置中的入口改为状态卡片。
- 保留启用开关、打开工作区和运行诊断三个原有操作。
- 设置入口与完整工作区共用同一视觉变量。

## 合并保护

rc.45 同时保留并验证：

- rc.42 名称锚点；
- rc.43 总结别名分发与 UI 调度；
- rc.42 链路收敛；
- rc.43 已覆盖小总结隐藏；
- rc.44 世界书同名托管条目 UID 归并。

## 自动验证

新增 `tests/rc45-ui-layout.ts`，检查：

- 桌面侧栏与主内容结构；
- 九个页面入口完整；
- 导航分组和当前页 `aria-current`；
- 移动端横向导航与单栏断点；
- 主内容、表格和编辑器滚动边界；
- 设置入口三个功能按钮不丢失；
- 活动小总结筛选仍生效。

静态容器中的 Chromium 被环境策略禁止访问本地预览页面，因此没有把容器截图列为视觉验收证据。最终视觉与触控体验仍需在真实 SillyTavern 桌面端和手机端各验证一次。
