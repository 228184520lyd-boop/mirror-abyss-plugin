# GitHub 网页上传说明

此版本专门用于 GitHub 网页端上传，文件数量低于 100。

操作方法：

1. 解压 `Mirror-Abyss-1.3.14-core.5-GitHub-under-100.zip`。
2. 打开解压后的目录。
3. 全选目录内的文件和文件夹。
4. 拖到 GitHub 仓库根目录的 `Upload files` 页面。
5. 提交更改。

仓库首页第一层必须直接看到：

- `manifest.json`
- `index.js`
- `style.css`
- `bootstrap/`
- `core/`
- `domain/`

不要把最外层 ZIP 作为唯一文件上传。

`docs/history/` 已收纳至 `_archives/docs-history.zip`，不影响插件运行。
