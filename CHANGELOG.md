# Changelog

## 1.0.0-rc.3.1

- 修复部分云酒馆/移动端顶部入口无法挂载或点击后无响应。
- 右下角“渊”按钮改为打开控制台，图谱仍可从控制台进入。
- 扩展设置区延迟出现时自动重试挂载。
- 控制台渲染失败时显示可见错误页，不再静默无响应。
- 无聊天或聊天元数据尚未就绪时，控制台仍可打开。
- 增加 `window.MirrorAbyss.openControlCenter()` 紧急入口。


## 1.0.0-rc.3

- 新增酒馆顶部镜渊入口与移动端全屏控制台。
- 控制台划分为总览、独立模型、表格与总结、规则校正、世界书与图谱、高级工具。
- 新增插件内独立 API：OpenAI 兼容、Gemini、Anthropic；支持模型列表读取与连接测试。
- 保留 Connection Profile 和当前聊天连接作为可选回退。
- 新增玩家自定义规则校正提示词。
- 规则未通过时可自动撤回最新 AI 消息，或保留并标记失败。
- 删除 `GENERATION_ENDED` 重复监听；新增消息级 Promise、定时去抖和正文指纹去重。
- 表格改为八分类标签 + 单一有边框工作簿，加入序号列和状态标签。
- 逐行编辑改为普通表单，不再要求直接编辑 JSON。
- 新增规则校正、表格整理、世界书同步的独立状态徽标与明确失败提示。
- 世界书同步增加 processing / success / error 状态和前端重试入口。
- 右下角浮动按钮默认关闭，可在高级设置中开启。

- 保留逐条编辑、删除和添加操作。
