# Changelog

## 1.2.0-rc.12

- 审核改为单次玩家提示词闭环：`MA_OK` 或 `MA_REVISED + 完整修正版`。
- 删除独立修正模型、二次审核、失败隐藏/标红和循环保护流程。
- 修正版由审核请求直接返回并原位替换，随后正常进入状态表、小总结和世界书流程。
- 升级时清理 rc.11 废弃的修正设置、修正连接和旧消息隐藏样式。
- 保留九表、分层总结、自动历史重建和世界书同步的既定功能。
- 修正文档中“历史不会自动重建”的过期说明。

## 1.2.0-rc.11

- 九张状态表采用独立字段模板和表头筛选。
- 加入焦点/人物去重、世界书保存校验与自动历史重建。
- 修复修正版正文以“修正版正文”等字样开头时可能被标题清理误删的解析边界。
- 修复工作区仍显示“不会自动重建历史”的过期文案。
