# Mirror Abyss / 镜渊

镜渊是 SillyTavern 前端扩展，提供玩家规则审核、九张结构化状态表、自适应分层总结和聊天世界书同步。

## 审核

- 插件读取玩家填写的审核提示词，并检查最新一条 AI 正文。
- 每条正文只调用一次审核模型。
- 返回 `MA_OK` 时保留原文。
- 返回 `MA_REVISED` 和完整修正版时，插件原位替换正文。
- 插件不额外增加审核标准，不使用本地关键词裁判，也不重新审核历史正文。

## API 调用

- “当前聊天连接”通过 SillyTavern `generateRaw` 调用。
- “Connection Profile”通过 `ConnectionManagerRequestService` 调用。
- 插件不保存 API 地址或密钥，也不直接连接模型供应商。

## 状态表与总结

- 九张状态表按各自表头提取和更新，允许本轮没有新增内容。
- 小总结根据已提取的状态变化决定触发时间；剧情变化密集时提前，变化较少时延后，但不会超过设置的最晚回合数。
- 大总结由若干小总结累积生成。

## 历史消息变化

编辑、删除或切换旧消息 swipe 后，镜渊暂停相关派生数据和世界书条目，并先尝试从变化位置自动重建。自动重建失败时，工作区保留手动选择起点重建的入口。历史重建只恢复状态表、总结和世界书，不审核或修改旧正文。
