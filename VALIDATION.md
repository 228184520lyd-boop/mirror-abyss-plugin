# Validation

版本：`1.1.0-alpha.6`

已完成：

- TypeScript `strict` 静态检查通过。
- ESM 生产构建通过。
- `node --check index.js` 通过。
- 27 项自动测试全部通过。
- 同一正文重复事件仍只创建一个模型任务。
- 审核 → 定向修正 → 再审核 → 状态提取顺序通过。
- 错误正文、审核说明和修正说明不新增为聊天消息。
- Connection Profile 任务会等待目标配置加载，并在完成后恢复原始“无 Profile”状态。
- 结构解析可处理说明文字、Markdown 围栏、思考标签、全角标点、尾随逗号和多个对象。
- 本地解析失败后仅调用一次格式修复，不重新执行原语义任务。
- 结构失败信息包含任务、连接配置和返回片段。
- 关系图谱缩放控件、范围输入、手机双向滚动和无最小 SVG 宽度限制通过。
- 人物生命周期、对象语义世界书、安全沉降、玩家锁定行和移动端横向表格原有测试继续通过。
- 静态确认插件设置中没有 API Key 字段。

尚未完成：

- 真实 SillyTavern 1.14 云部署中多个厂商 Profile 连续切换的实机回归。
- Connection Manager 被停用、Profile 在任务中被删除或第三方插件同时切换连接时的冲突测试。
- 真实模型返回超长截断 JSON、非标准 JSON5 或工具调用对象时的兼容性测试。
- 关系图谱在数百节点下的性能和触控缩放手势测试；本版提供滑杆与按钮缩放，不宣称支持双指手势。
- 流式输出、群聊、Swipe、消息编辑与删除组合测试。
- 100 回合以上世界书沉降与长期存档测试。

因此本版本仍标记为 alpha，不宣称长期稳定版。
