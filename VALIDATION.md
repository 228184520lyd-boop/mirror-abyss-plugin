- 结构化审核与旧 `MA_OK / MA_FAIL` 兼容解析通过。

- 自定义主题文字与面板变量隔离通过。
- 静态确认未监听 `GENERATION_ENDED` 执行业务任务。
- 静态确认插件设置中没有 API Key 字段。
